> Sorting Randomized Images_
This is a Python command-line program that helps with the visualization of sorting algorithms. It does this by taking an image (png, jpg, etc.) and randomizing it based on every column of pixels in the image. For example, if the image resolution is 1280x720, there will be 1280 columns of pixels in the image; this is what will be randomized. The program then employs a user-chosen sorting algorithm that re-sorts the image back to its original state. A video is then output that shows, step-by-step, the process that the algorithm utilized to re-sort the image. My program is unique in the way that it allows users to input any picture they desire so that it can be randomized and sorted; this adds a “fun” factor when learning these algorithms.
> Birds Hate Bombs_

Birds Hate Bombs is an infinite runner game made using the Unity game engine. You control a bird determined to avoid the gravity-defying bombs that litter his environment. The original version of this game was my submission to the University of Oklahoma Game Developers Association Fall 2019 Game Jam. The theme for this Jam was "two buttons," which I satisfied by only having two buttons that control player movement.

> An Application of Simple Linear Regression_
This is my final project for my Applied Statistics course I took in during the Spring 2019 semester. It is an R Markdown document of my statistical analysis of data I collected while playing the popular video game Fortnite. More specifically, I used simple linear regression to analyze the damage dealt to opponents and the number of eliminations per game in order to determine whether there was a linear relationship between those two in-game statistics. I used R to perform the statistical tests and to make my conclusion.
